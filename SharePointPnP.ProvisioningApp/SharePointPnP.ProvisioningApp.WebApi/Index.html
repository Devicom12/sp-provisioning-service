SharePoint PnP Provisioning Service API

This is the entry point for the multi-tenant SharePoint PnP Provisioning Service API. Here you can find two different kind of API endpoints:

  • Catalog API: provides access to provisioning packages (templates), and categories. Requires OAuth App-Only authentication through Azure Active Directory. Supports OData v. 4.0 protocol and is available at the following URLs:
    • Categories OData endpoint
    • Packages OData endpoint
  • Provisioning API: provides access to the provisioning service. Requires OAuth delegated access tokens authentication through Azure Active Directory. Is document at the following URL: Swagger Documentation.
